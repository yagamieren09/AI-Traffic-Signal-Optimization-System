# 🚦 AI-Based Multi-Intersection Traffic Signal Optimization

## Quick Start — Dashboard

```bash
pip install numpy matplotlib flask
python server.py
# Open http://localhost:5000
```

### Dashboard has 3 tabs:
- 🧠 Train Agent — live training charts, reward/queue curves
- 🚦 Live Simulation — watch 4 intersections, switch RL ↔ Fixed-Time
- 📊 Results — comparison metrics & bar charts

## CLI Only
```bash
pip install numpy matplotlib
python main.py
```

- Wait Time: +34.9% improvement
- Queue Length: +76.2% improvement
